# gitetc-rm(1) -- un-track an /etc file

## SYNOPSIS
`gitetc-rm` `-h`|`--help`<br>
`gitetc-rm` _file_ ...<br>

## DESCRIPTION
`gitetc-rm` removes a previously tracked `/etc` file from the backing
repository.

## OPTIONS
* -h, --help:
  Show help text and exit.
